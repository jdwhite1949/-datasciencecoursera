Health and Economic Effects of Storm Events

Jim White

October 21, 2015

Synopsis

This report makes use of a dataset from the National Weather Service & National Climatic Data Center Storm Events. The construction and definition of the variables in the data can be found at Storm Data Documentation and FAQs from the Climatic Data Center. The purpose of this analysis was to determine the weather events that have the most effect on health (fatalities and injuries) and economy (property and crop damage). The event types having the greates impact on fatalities for the period between 1996 and 2011 were excessive heat, tornados, and flash floods. The greatest number of injuries are due to tornados. Not surprisingly, hail creates the greatest amount of economic loss for crops, followed by flooding. The largest contributors to property loss are thunderstorm winds, flooding, and tornados.

Loading and Processing the Data

Data Description The original dataset was a csv file with 902,297 observations and 37 variables and represents the years from 1950 to 2011. Prior to 1996, the data collected was not representative of the 48 event types as defined by the NWS Directive 10-1605, consequently years prior to 1996 will be ommitted from the analysis (they would not be comparable). The latest date in the dataset is 11-30-2011. Recent years may represent more relevant information. The transformed dataset used for the analysis represents 653,529 observations with 7 variables.

The variables included in the transformed dataset are:

1. EVTYPE - type of meterological event (48 different types available)
2. END_DATE - the end date of the event
3. FATALITIES - the number of deaths related to the event (individuals)
4. INJURIES - the number of injuries related to the event (individuals)
5. PROPDMG - estimated amount of damage to property due to the event (thousands of dollars)
6. CROPDMG - estimated amount of damage to crops due to the event (thousands of dollars)

Requirements This analysis requires the use of these packages: dplyr, gridExtra, ggplot2

Preliminaries - loading necessary libraries

suppressMessages(library(dplyr))
library(gridExtra)
library(ggplot2)

Data Processing First download the file and load into R as activity_data variable. Only load those columns (variables) relative to answering the data analysis questions. This part of the process will takes a few minutes because of the size of the original data file.

download.file("https://d396qusza40orc.cloudfront.net/repdata%2Fdata%2FStormData.csv.bz2", "stormData.bz2")
activity_data <- read.csv("StormData.bz2")[, c(8, 12, 23, 24, 25, 27)]

Transform EVTYPE & END_DATE variables to character and date types, respectively, and PROPDMG & CROPDMG to integer format

activity_data$EVTYPE <- as.character(activity_data$EVTYPE)
activity_data$END_DATE <- as.Date(activity_data$END_DATE, "%m/%d/%Y ")
activity_data$PROPDMG <- as.integer(activity_data$PROPDMG)
activity_data$CROPDMG <- as.integer(activity_data$CROPDMG)

Remove records with END_DATE = NA and any date < year 1996 (see explanation above about events prior to 1996)

activity_data_sub <- subset(activity_data, !is.na(END_DATE) & END_DATE > as.Date("1995-12-31"))

Group the dataset by the ENVTYPE variable and sum the fatalities, injuries, property, and crop damage variables to find the EVTYPES that create the greatest health and economic effects.

# summarize and grouped by EVTYPE
summary_data <- activity_data_sub %>% group_by(EVTYPE) %>% summarize(fatal = sum(FATALITIES), injury = sum(INJURIES), propdmg = sum(PROPDMG), cropdmg = sum(CROPDMG))

Each of the four variables are subset as a preparation for providing charts.

# subset fatalities and sort decreasing
fatal_events <- summary_data[, c(1, 2)]
